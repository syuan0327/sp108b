# cargo 
## 使用方法
#### 一、
```
cargo new hi
```
(*註：此處的hi為RUST的工程目錄名稱，所以可以任意取名)

打好後會出現名為hi的工程目錄

目錄：

<img src = "https://github.com/syuan0327/sp108b/blob/master/rust/cargo/hi/1.JPG" width=50% height=50%>

#### 二、
```
cd ./hi
cargo build 
cargo run 
```
cd ./hi => 進到hi目錄

cargo build => 系統在創建工程時會生成一個 Hello, world 來源程式 main.rs
